 Pull چیست؟

Pull یعنی گرفتن آخرین تغییرات از مخزن آنلاین (GitHub) و اعمال آن روی پروژه محلی.

اگر شخص دیگری روی پروژه تغییر ایجاد کرده باشد، با Pull کردن، پروژه‌ی شما به‌روز می‌شود.

Pull در واقع شامل دو مرحله است:
1. دریافت تغییرات (fetch)
2. ادغام تغییرات با پروژه محلی (merge)
