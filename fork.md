Fork چیست؟

Fork یعنی ساختن یک کپی کامل از مخزن (Repository) یک پروژه در حساب GitHub خود

با Fork کردن یک پروژه یک نسخه مستقل از آن در اکانت گیت هاب ساخته میشود که میتوانیم بدون تغییر دادن پروژه اصلی روی آن تغییرات بدهیم

Fork معمولاً زمانی استفاده می‌شود که:
- صاحب پروژه نیستی
- می‌خواهی تغییر ایجاد کنی
- قصد داری بعداً تغییراتت را با Pull Request به پروژه اصلی ارسال کنی
