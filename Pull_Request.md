 Pull Request چیست؟

Pull Request یا PR یعنی درخواست رسمی برای اضافه شدن تغییرات شما به پروژه اصلی.

وقتی روی یک Fork یا Branch تغییر ایجاد می‌کنید و می‌خواهید صاحب پروژه آن را بررسی و قبول کند، Pull Request ارسال می‌کنید.

Pull Request امکان‌های زیر را می‌دهد:
- بررسی کد  
- گذاشتن کامنت  
- رد یا قبول تغییرات  
- Merge کردن کدها
