تفاوت Pull، Pull Request و Push

- Pull  
  دریافت آخرین تغییرات از GitHub به سیستم محلی. فقط پروژه محلی شما را به‌روز می‌کند.

- Push  
  ارسال تغییرات از سیستم محلی به GitHub. تغییرات شما را روی مخزن آنلاین ثبت می‌کند.

- Pull Request  
  درخواست برای ادغام تغییرات شما در پروژه اصلی. شامل بررسی، کامنت و تایید صاحب پروژه است.


- Push فقط تغییرات را ارسال می‌کند  
- Pull فقط تغییرات را دریافت می‌کند  
- Pull Request یک فرآیند بررسی و ادغام است که توسط صاحب پروژه تایید می‌شود
